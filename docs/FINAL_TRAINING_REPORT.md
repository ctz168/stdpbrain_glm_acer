## 四、关键发现

### 1. 灾难性遗忘问题

```
不平衡训练导致:
- 语言能力提升: +33.3%
- 逻辑能力下降: -66.7% ⚠️
- 常识能力下降: -33.3% ⚠️
```

### 2. 解决方案

| 方法 | 效果 |
|------|------|
| 平衡训练数据 | ✅ 防止遗忘 |
| 更小学习率(5e-6) | ✅ 稳定训练 |
| 90%权重冻结 | ✅ 保护基础能力 |
| 梯度裁剪(0.5) | ✅ 防止梯度爆炸 |

### 3. 最佳实践

```python
# 推荐配置
config = {
    'epochs': 1,           # 少轮次
    'learning_rate': 5e-6, # 小学习率
    'weight_freeze': 0.9,  # 冻结90%
    'data_balance': True,  # 平衡数据
    'grad_clip': 0.5       # 梯度裁剪
}
```

### 主要结论

1. ✅ **模型基础能力强**: 数学、逻辑、常识均达100%
